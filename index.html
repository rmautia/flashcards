Definitions

  • Javascript

    an object-oriented computer programming language commonly used to create interactive effects within web browsers.

  • Operators

    The definition of an operator is a symbol that is used to perform an operation.

  • Variables

    A defining variable is a symbol, such as x, used to describe any number. When a variable is used in an function, we know that it is not just one constant number, but that it can represent many numbers. Variables are instrumental in understanding problems relating to graphing.

  • Functions

    A function is a subprogram designed to perform a particular task. Functions are executed when they are called. This is known as invoking a function.
